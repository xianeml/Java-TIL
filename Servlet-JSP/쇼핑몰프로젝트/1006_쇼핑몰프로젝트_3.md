### MemberService:

```
public int memberUpdate(MemberDTO dto) {
	SqlSession session = MySqlSessionFactory.getSession();
	int n = 0;
	try {
		MemberDAO dao = new MemberDAO();
		n = dao.memberUpdate(session, dto);
		session.commit();
	} catch (Exception e) {
		e.printStackTrace();
	} finally {
		session.close();
	}
	return n;
}// end memberUpdate
```
